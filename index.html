Q1 Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype

ans: 
        const parentObj = {
            calAge : function(){
                console.log(`${this.fname} ${this.lname} ${this.age}`)
            }
        }

        const childObj = Object.create(parentObj);
        childObj.fname = "EA18";
        childObj.lname = "2022";
        childObj.age = "Oct Batch";

       const childObj = Object.create(parentObj);
        childObj.fname = "John";
        childObj.lname = "Mathew";
        childObj.age = 20;

        childObj.calAge();
     OutPut: 
     EA18 2022 Oct Batch

     John Mathew 20

Q2 Write code to explain prototype chaining

ans: 
        Prototype chaining is used to build new types of objects based on existing ones. It is similar to inheritance in a class based language.

        class Person {
            constructor() {
              this.gender = 'male';
            }
            printPerson() {}
          }
          class User extends Person {
            constructor() {
              super();
              this.name = 'Abhraham';
            }
            printUser() {}
          }
          var u = new User();
          console.log(u);

 output:
     User {gender: 'male',name: 'Abhraham'}

Q3 Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays
